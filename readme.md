# Task Manager  
A simple task manager app built using React and Redux.  

### Note:  
- This project uses Vite.js as the front-end build tool.  
- The project was bootstrapped using the command `npm init vite`.  
- This app uses Redux store for state management.

## Features  
- Add new tasks.  
- Delete tasks.  
- Filter tasks.  
- Mark tasks as completed and toggle their status.  

## Technologies Used  
- HTML  
- CSS  
- JavaScript  
- Node.js  
- React  
- Redux  

## Installation and Setup  
1. Clone the repository to your desired location.  
2. Open the project in your code editor.  
3. Install all dependencies by running the following command in the terminal:  
   npm install  
   npm run dev
